AIIA helps teams using JIRA by understanding your project history

Optimise decisions
Save time

Remain consistent



AIIA is a machine learning plugin developed for the JIRA ecosystem. This plugin can suggest optimal human resource assignment, task componenents, task labelling and story point estimates for any task as requested. In the future, the plugin is to be extended to add visualization of issue dependencies and predictive analytics of issue dependencies.

End User

The primary target market for this product shall be JIRA users who are:

  • Part of small to medium-sized enterprises - As they make up most enterprises using JIRA, and most likely posses the least resources, therefore an AI helper would be of more aid to their organisation.
  • Scrum development team members – Developers who are using JIRA as a management tool shall be able to use this product to aid their decision-making process.
  • Project managers / Scrum masters – Project managers or Scrum masters may use this product to confirm or review their own conclusions in the areas of assistance, thus decreasing the time spent on these tasks.
  • English-speaking – Appeals to most JIRA users, and is a language spoken by all members of the development team for the product.
  • Predictive analytics – Is a niche market within the Atlassian marketplace.

Design Approach

The design methodology that has been engaged within the viTech team Kanban focused and aims to support the idea of changing schedules within the development team. As the project progresses further through development, the efficiency and effectiveness of this approach has improved over time. The team strives for a safe development environment to allow for the expressions to be freely discussed and to help provide an area for innovation.

Development Environments

  • JIRA(ACE)
    • The Atlassian Connect Express (ACE) framework is constructed in Node.js and aims to provide a simple yet effective way to create a plugin for the JIRA ecosystem. From this, interface elements can be implemented using AUI or Atlaskit to provide the user with a portal for powerful functionality at just a few clicks away.
  • Python
    • Powerful machine learning libraries have been engaged with in the Python language and are used to process data to create the core functionality of this JIRA plugin. The microframework Flask developed for Python has been used to implement a REST-ful interface for the ACE framework to communicate with.
